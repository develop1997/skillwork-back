# skillwork-back

backend for [skillwork](https://github.com/develop1997/skillwork), this project is using [express](https://expressjs.com/) and [firebase](https://firebase.google.com/) to conect the users.

## Get started

1. Install dependencies

    ```bash
    pnpm install
    ```

2. Run the server

    ```bash
    pnpm start
    ```
